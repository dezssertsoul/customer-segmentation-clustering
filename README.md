Machine Learning Clustering & Classification Project

*Deskripsi Proyek*
Proyek ini merupakan bagian dari perjalanan saya sebagai AI Engineer Cohort di Pijar dan diselesaikan melalui kurikulum Dicoding. Fokus utamanya adalah menerapkan teknik unsupervised learning untuk pengelompokan data, yang kemudian divalidasi menggunakan algoritma klasifikasi.

*Alur Kerja (Workflow)*
a. Preprocessing: Pembersihan data, scaling, dan encoding fitur.
b. Dimensionality Reduction: Menggunakan PCA untuk efisiensi fitur tanpa menghilangkan informasi penting.
c. Clustering: Pengelompokan data untuk membentuk label target.
d. Supervised Learning: Melatih model Decision Tree untuk memprediksi hasil clustering pada data baru.
e. Evaluasi: Menggunakan Classification Report (Precision, Recall, F1-Score) untuk memastikan performa model yang akurat.

*Teknologi yang Digunakan*
a. Python
b. Pandas & NumPy
c. Scikit-Learn
d. Matplotlib/Seaborn
